# Nyaa.si Batch

- Download multiple `.torrent` files.
- Optional ability to look for closest matching titles.
- Prioritizes torrents with more seeders.

### Prerequisites

* Python 3.9+ (3.0+ might work)

### Usage

```
python main-cli.py --help
```
